## 3. 题目本质

这题本质上是在问�?

> 对于一个不断向右滑动的固定长度窗口，怎样快速维护窗口中的最大值？

## 4. 暴力解法

## 4.1 思路

每形成一个窗口，就遍历窗口中�?`k` 个元素，找最大值�?

---

## 4.2 代码（C语言�?

```c
int* maxSlidingWindow(int* nums, int numsSize, int k, int* returnSize) {
    *returnSize = numsSize - k + 1;
    int* ans = (int*)malloc(sizeof(int) * (*returnSize));

    for (int i = 0; i <= numsSize - k; i++) {
        int maxVal = nums[i];
        for (int j = i; j < i + k; j++) {
            if (nums[j] > maxVal) {
                maxVal = nums[j];
            }
        }
        ans[i] = maxVal;
    }

    return ans;
}
```

---

* 时间复杂度：`O((n-k+1) * k)`，最坏可写作 `O(nk)`
* 空间复杂度：`O(1)`（不算返回数组）

## 9. 如何维护单调递减队列

当遍历到新元�?`nums[i]` 时，做两件事�?

### 9.1 移除队首中过期元�?

## 10. 算法流程

遍历数组下标 `i`�?

1. 移除队首中过期的下标
2. 从队尾删除所有值小于等于当前值的下标

## 11. C语言标准实现

```c

/**
 * Note: The returned array must be malloced, assume caller calls free().
 */
int* maxSlidingWindow(int* nums, int numsSize, int k, int* returnSize) {
    *returnSize = numsSize - k + 1;
    int* result = (int*)malloc(sizeof(int) * (*returnSize));
    int* deque = (int*)malloc(sizeof(int) * numsSize);  // 存下�?

    int front = 0, rear = -1;

    for (int i = 0; i < numsSize; i++) {
        // 1. 移除队首过期元素
        while (front <= rear && deque[front] <= i - k) {
            front++;
        }

        // 2. 保持队列单调递减
        while (front <= rear && nums[deque[rear]] <= nums[i]) {
            rear--;
        }

        // 3. 当前元素下标入队
        deque[++rear] = i;

        // 4. 窗口形成后，记录最大�?
        if (i >= k - 1) {
            result[i-k+1] = nums[deque[front]];
        }
    }

    free(deque);
    return result;
}
```
