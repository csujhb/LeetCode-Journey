2. * # 19. 删除链表的倒数第 N 个结点
   
     ## 题目描述
   
     给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。
   
     ## 示例
   
     **示例 1：**
   
     > **输入：** head = [1,2,3,4,5], n = 2
     >
     > **输出：** [1,2,3,5]
   
     **示例 2：**
   
     > **输入：** head = [1], n = 1
     >
     > **输出：** []
   
     **示例 3：**
   
     > **输入：** head = [1,2], n = 1
     >
     > **输出：** [1]
   
     ## 提示
   
     - 链表中结点的数目为 `sz`
     - $1 \le sz \le 30$
     - $0 \le Node.val \le 100$
     - $1 \le n \le sz$
   
     ## 进阶
   
     你能尝试使用一趟扫描实现吗？思考如何利用双指针（快慢指针）维持一个长度为 `n` 的窗口，从而在快指针到达末尾时，慢指针正好指向待删除节点的前驱位置。
