## 5. C 语言实现

```c

    for (int i = 0; i < 26; i++) {
        if (a[i] != b[i]) {
            return 0;
        }
    }

    int sLen = strlen(s);
    int pLen = strlen(p);

    }

            result[(*returnSize)++] = i - pLen + 1;

    }

    return result;
}
```
