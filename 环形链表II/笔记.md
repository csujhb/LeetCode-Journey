## 二�?LeetCode 142. 环形链表 II（寻找环的入口）

### 💡 核心逻辑与数学推�?

**变量定义�?*

**🌟 结论释义�?*

### 💻 结构化代码模�?

```c
    struct ListNode *slow = head, *fast = head;
    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
        
        if (slow == fast) {

            while (ptr != slow) {
                ptr = ptr->next;
                slow = slow->next;
            }
            return ptr; // 再次相遇点即为入�?
        }
    }
    return NULL; 
}
```
